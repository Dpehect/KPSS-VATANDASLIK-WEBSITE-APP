# KPSS Vatandaşlık Web Application - Advanced Architecture & Study Engine

- GitHub Repository: https://github.com/Dpehect/KPSS-Vatandaslik-Web-App
- Live Demo: https://kpss-vatandaslik-web-app.vercel.app/

This project is a high-performance, enterprise-grade educational web application designed for the KPSS Citizenship (Vatandaşlık) exam preparation. It features a massive database of questions, interactive level-based tests, flashcards for active recall, and dynamic topic summaries on public law, civil rights, and constitutional history.

## Core Architectural Pillars

- **Framework**: Next.js utilizing App Router and Turbopack for optimized build performance.
- **Language**: TypeScript for complete compile-time type safety across all components and data structures.
- **Database & Sync**: Supabase with SSR and custom seeding, backed by a local-first fallback mechanism for zero-latency loading.
- **Styling & Layout**: Custom utility system built on top of Tailwind CSS v4, supporting customized dataset dark mode variants (`data-theme="dark"`).
